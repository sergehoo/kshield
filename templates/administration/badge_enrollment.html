⚙️ Comment connecter mon lecteur ?
  • USB / clavier-émulé (ACR122, OMNIKEY, RFIDeas pcProx) : le lecteur tape l'UID puis Entrée — focalisez la textarea "Saisie batch" et passez chaque carte.
  • Lecteur IP UHF (Impinj/Zebra) : configurer le lecteur pour envoyer chaque tag à POST /api/v1/devices/scan/inbox/ avec body {"reader_id": <serial>, "uid": "AABBCC01"}. Le lecteur doit être enregistré (page équipements) pour être autorisé.
  • App mobile (BLE scan) : utiliser l'app KAYDAN SHIELD pour scanner les beacons BLE alentours.
